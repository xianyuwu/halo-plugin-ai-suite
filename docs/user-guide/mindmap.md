# AI 脑图

> 适用读者：Halo 站长、内容运营人员

AI 脑图把文章结构生成 Markdown 树并缓存到文章 annotation，访客端再渲染为可折叠、可跳转的交互脑图。

![AI 脑图管理](../../assets/readme/screenshots/console-mindmap.jpg)

## 生成流程

1. 开启脑图功能。
2. 设置温度、最大输出、输入字符上限和层级深度。
3. 选择文章生成、重新生成或批量生成。
4. 在后台预览结构。
5. 打开文章页检查主题适配、折叠和原文跳转。

![文章页 AI 脑图](../../assets/readme/screenshots/visitor-mindmap.jpg)

## 配置建议

- 温度保持在 0.2～0.4，结构会更稳定。
- 深度建议 3；过深会增加噪声和输出失败概率。
- 超长文章使用输入上限控制费用，但要接受尾部内容可能不进入生成。
- 自定义要求用于约束节点长度和重点，不要要求模型输出复杂样式。

## 缓存与过期

缓存同时关联文章内容和脑图配置。文章或关键生成配置变化后，旧结果可能被标记为 stale。重新生成会覆盖缓存；清除只移除脑图，不删除文章。

## 批量任务

“生成全部”会创建后台任务，可查询进度并取消。生产环境先用少量文章验证模型输出，再启动全量任务。

## 验证

- 节点能够展开和折叠。
- 点击节点文字能跳转到对应原文位置。
- 亮色和暗色主题均可读。
- 移动端保持文章上下文，不出现强制全屏遮挡。
- 文章更新后能够识别陈旧缓存。
